## Installation

Python 3.10+ is supported. Install all requirements, preferably using Poetry(https://python-poetry.org/):
* Clone this repo
* Install poetry:
```bash
python3 get-pip.py
curl -sSL https://install.python-poetry.org | python3 -
```
* Setup Poetry:
```bash
poetry install
```
* Run the webapp:
```bash
poetry shell
streamlit run currency_converter.py
```
